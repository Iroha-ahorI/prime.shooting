# Prime Shooter JS版 Python(Pygame)版をブラウザで遊べるように移植した作品です。 ## 操作方法 - ← → : プレイヤー移動 - ↑ : 弾を発射 - ↓ : 弾をスキップ ## ルール - 敵の数字を割り切れる弾を当てると敵が分解されます。 - 敵が消えた位置に応じてスコア加算。 - 割り算成功でも敵が残ればスコア変動なし。 - 割り算失敗なら -20 点。 - 敵が下まで到達するとライフ減少。 - ライフが 0 → GAME OVER。 - スコアが 1000 → GAME CLEAR。 # Prime Shooter (JavaScript Edition) A browser-based shooting game where you fire number bullets to break down composite enemies. ## Controls - **← / →** : Move the player left and right - **↑** : Shoot the current bullet - **↓** : Skip the current bullet (discard and replace) ## Rules 1. **Enemy division** - If the enemy's number can be divided by the bullet's number, the enemy is reduced. - If the enemy becomes `1`, it disappears. 2. **Scoring** - If the enemy disappears, you gain points based on its position: `Score += (HEIGHT - enemy.y) / 10` - If the division succeeds but the enemy remains, **no score change**. - If the division fails, **-20 points**. 3. **Lives** - If an enemy reaches the bottom (player line), you lose 1 life. - You start with **5 lives**. - If lives reach 0 → **GAME OVER**. 4. **Game Clear** - If your score reaches **1000 points**, you win → **GAME CLEAR**. 5. **Visual Feedback** - When scoring changes, a temporary popup (`+points` or `-20`) appears near the collision point for 0.5 seconds. ## Goal Break down composite numbers using prime bullets, survive, and aim for the highest score!
